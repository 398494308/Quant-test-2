# 激进版 MACD 当前设定

定位
- 面向高风险高回报场景
- 目标是让月度收益弹性明显高于保守版
- 接受 20%-30% 级别回撤和有限爆仓概率

核心参数
- 杠杆：15x
- 并发仓位：4
- 单仓资金占比：20%
- 第一止盈：7.0%
- 第一止盈平仓比例：35%
- trailing 激活：12.0%
- trailing 回撤容忍：6.0%
- 基础持仓：35 根 15m
- 动态持仓上限：45 根 15m

信号逻辑
- breakout 更宽，尽量抓趋势启动
- pullback 也更宽，但仍保留 EMA/ADX/MACD 结构
- 小时级过滤仍保留，只是阈值明显放松

高级功能
- 启用一次金字塔加仓
- 盈利达到 6% 且 ADX 足够强时加仓
- 默认最多加仓 1 次，加仓比例为原仓位 50%
