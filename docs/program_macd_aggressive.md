# 激进版 MACD 策略优化目标

## 交易框架
- 标的：BTCUSDT 永续合约
- 杠杆基准：15x
- 执行周期：15分钟
- 趋势过滤周期：1小时
- 只做多，不做空
- 默认最多 4 个并发仓
- 单仓目标保证金占账户约 20%
- 单仓保证金下限 5000 USDT
- 单仓保证金上限 30000 USDT

## 策略目标
- 目标不是平滑，而是提高月度爆发力
- 接受更高回撤和一定爆仓风险
- 尽量抓住大级别趋势段，让利润跑得更久
- breakout 负责抓趋势启动
- pullback 负责趋势续涨

## 允许的激进方向
- 放宽 breakout / pullback 的 ADX、MACD、Volume 条件
- 放宽 1h 趋势环境过滤
- 延后止盈，拉宽 trailing
- 提高并发和仓位利用率
- 允许金字塔加仓

## 验证要求
- 优先看滚动月度窗口总收益和夏普/卡玛
- 同时记录：
  - 最大回撤
  - 爆仓次数
  - 平均持仓
  - 月度收益分布
- 激进版不要求像保守版那样严格，但要能清楚暴露风险

## 约束
- 只允许改参数块，不要重写策略框架
- leverage、tp1、trailing、pyramid 参数都允许优化
- 不要把交易数压到过低
- 不要为了单月暴冲而完全牺牲后续月份
